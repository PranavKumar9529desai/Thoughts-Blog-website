# Project Title

This is a full-stack application built with React, TypeScript, and Vite on the frontend, and Node.js on the backend. The application uses Recoil for state management and Prisma as the ORM for PostgreSQL.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Node.js
- npm
- PostgreSQL
